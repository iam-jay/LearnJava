30-Day Java Learning Plan

This comprehensive plan will guide you through essential Java concepts, starting from the basics in Java 8 to the advanced features of Java 21.

Phase 1: Java 8 Core Concepts (Days 1-10)

  • Day 1: OOP Concepts - Oracle Java Tutorials - OOP
  • Day 2: Encapsulation and Abstraction - GeeksForGeeks
  • Day 3: Collections Framework - Baeldung
  • Day 4: Exception Handling - JavaPoint
  • Day 5: Input and Output (I/O) - JavaPoint
  • Day 6: Threads - Oracle Java Tutorials
  • Day 7: Lambda Expressions - Baeldung
  • Day 8: Stream API - Baeldung
  • Day 9: Functional Interfaces and Optional - GeeksForGeeks, Baeldung
  • Day 10: Date/Time API - Oracle Java Tutorials

Phase 2: Java 11 Features (Days 11-20)

  • Day 11: Strings Enhancements - Baeldung
  • Day 12-13: HTTP Client - Baeldung
  • Day 14: Local Variable Syntax for Lambda - GeeksForGeeks
  • Day 15-16: Streams Enhancements - Baeldung
  • Day 17-20: Practice Project - Build a mini REST client. FreeCodeCamp

Phase 3: Java 21 Modern Features (Days 21-30)

  • Day 21-22: Virtual Threads - Baeldung
  • Day 23-24: Record Patterns - Oracle - Java 21 Features
  • Day 25-26: Sequenced Collections - Baeldung
  • Day 27-28: String Templates - Baeldung
  • Day 29-30: Final Project - Build a multithreaded REST API using Virtual Threads. Java 21 Tutorials
